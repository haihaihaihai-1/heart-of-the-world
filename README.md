# 🌍 《世界之心》— Heart of the World

> 一款 Minecraft 风格的 3D 体素冒险游戏，讲述「苏醒者」寻找破碎的世界之心碎片、揭开身世之谜、对抗「虚无」的热血冒险故事。

## ✨ 游戏特色

- 🎮 **完整 3D 体素世界**：第一人称探索无限程序生成的地形
- ⛏️ **方块破坏与放置**：挖矿、建造，原汁原味的 Minecraft 玩法
- 🌲 **程序生成地形**：草原、森林、沙漠、雪原四种生物群系，树木、矿石、洞穴、水系齐全
- 📖 **丰富剧情**：序章 + 第一章完整可玩，多章节故事线数据驱动可扩展
- 🧙 **NPC 与对话系统**：与「艾尔登长老」交互，逐字显示的剧情对话
- 🎯 **任务追踪**：屏幕右侧实时显示当前任务进度
- 💎 **特殊道具**：发光的「世界之心碎片」触发剧情演出
- 🎨 **零素材依赖**：所有贴图用 Canvas 程序化生成（16×16 像素风），开箱即用

## 🚀 快速开始

### 方法一：网页版（最简单，推荐）

1. **安装 Node.js**（如已安装可跳过）：访问 https://nodejs.org/ 下载 LTS 版本
2. 双击目录中的 **`运行游戏.bat`**
3. 首次运行会自动安装依赖（约 1-3 分钟），之后会自动打开浏览器
4. 看到「进入世界」按钮后点击即可开始游戏

或者手动用命令行：
```bash
npm install      # 安装依赖
npm run dev      # 启动开发服务器
```

### 方法二：Electron 桌面应用

```bash
npm install
npm run electron     # 会先打包再用桌面窗口打开
```

或双击 **`运行桌面版.bat`**。

## 🎮 操作说明

| 按键 | 功能 |
|------|------|

## 📖 故事背景

远古时代，世界由一颗「**世界之心**」维系生机。某日，黑暗领主「**虚无**」击碎了世界之心，碎片散落四方。世界开始崩塌——怪物横行，大地荒芜。

你是一位从远古沉睡中苏醒的「**苏醒者**」。300 年前，守望者艾尔登长老预言你的归来。如今，你肩负着重铸世界之心的使命。每找到一块碎片，你失去的记忆就会复苏一些……

### 当前可玩内容

- **🌅 序章：苏醒之地** — 苏醒、遇长老、学砍树、收集木材（教学）
- **⚔️ 第一章：森林的试炼** — 进入古老森林，寻找第一块世界之心碎片
- **🚧 后续章节** — 数据已预留，可平滑扩展（沙漠、雪山、地底、虚无之战）

## 🛠️ 技术栈

| 层面 | 技术 | 作用 |
|------|------|------|
| 3D 渲染 | [three.js](https://threejs.org/) | WebGL 体素渲染 |
| 构建工具 | [Vite](https://vitejs.dev/) | 极速开发服务器与打包 |
| 桌面打包 | [Electron](https://www.electronjs.org/) | 把网页包装成桌面应用 |
| 地形噪声 | [simplex-noise](https://www.npmjs.com/package/simplex-noise) | 自然地形生成 |
| 贴图 | 原生 Canvas API | 程序化生成像素贴图（零素材依赖） |

### 性能优化

- ✅ **面剔除**：只渲染暴露在空气中的方块面（省 80% 三角形）
- ✅ **纹理图集**：所有方块共用一张贴图，单次绘制调用
- ✅ **圆形渲染范围**：以玩家为中心的 4 区块半径（约 81 个区块）
- ✅ **视锥剔除**：Three.js 自动剔除屏幕外的区块
- ✅ **方向光照烘焙**：用顶点色实现立体感，无需实时光照计算

## 🔧 常见问题

**Q: 启动后画面是黑的 / 没反应？**
A: 等待几秒，区块正在生成。也可以打开浏览器开发者工具（F12）查看是否有报错。

**Q: 帧数很低、卡顿？**
A: 这通常是因为视野内有太多方块。可以编辑 `src/game/World.js` 中的 `RENDER_RADIUS`（默认 4）改小一些。

**Q: 鼠标锁不住、点画面没反应？**
A: 确保点击的是游戏画布区域。如果还是不行，刷新页面重试。

**Q: 想修改剧情/任务？**
A: 编辑 `src/story/Chapters.js`（剧情文本）和 `src/story/Quests.js`（任务数据）即可，无需改代码逻辑。

**Q: 想换个世界？**
A: 编辑 `src/game/Game.js` 第 17 行的 `SEED = 1337`，改成任意数字就会生成不同的世界。
